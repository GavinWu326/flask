关于 Halowork 项目

🎯 项目目标

Halowork 是一个专为测试 CI/CD (持续集成/持续部署) 流程而设计的 Flask Web 应用程序。 它提供了完整的项目结构和必要的接口，帮助开发者验证自动化部署流程。

✨ 主要特性

  • 健康检查接口: /health - 用于监控应用状态
  • 应用信息接口: /api/info - 提供版本和环境信息
  • 响应式界面: 基于 Bootstrap 5 的现代化 UI
  • 环境变量支持: 便于容器化和云部署
  • 标准项目结构: 遵循 Flask 最佳实践

🛠️ 技术栈

  • Python 3.x
  • Flask 框架
  • Jinja2 模板引擎
  • Bootstrap 5
  • RESTful API
  • JSON 响应

🚀 部署信息

此应用支持多种部署方式，包括传统服务器部署、Docker 容器化部署， 以及云平台部署（如 AWS、Azure、Google Cloud）。

提示: 访问 /api/info 接口可以获取当前部署的详细信息。
